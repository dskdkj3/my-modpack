# Prominence II Hasturian Era client profile

This branch contains a Packwiz client profile for Prominence II Hasturian Era 3.9.26 with Distant Horizons added.

- Minecraft: 1.20.1
- Loader: Fabric 0.18.4
- Shader stack: the official Prominence II client pack already includes Iris/Sodium and Complementary Reimagined r5.7.1.
- Extra client mod: Distant Horizons 3.0.2-b for 1.20.1 Fabric/Forge.

Import `client/pack.toml` in a Packwiz-capable launcher or serve this profile with Packwiz for HMCL-style import.
